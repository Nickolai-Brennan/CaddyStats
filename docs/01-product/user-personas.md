# User Personas

Last updated: 2026-05-12
Owner: Product Research
Status: Working draft

## Objective

Define the primary user personas for Caddy Stats so product, content, pricing, and retention decisions are grounded in real usage patterns.

## Persona 1: Data-Driven Golf Bettor

### Profile

- Age: 25-45
- Behavior: Bets weekly golf markets
- Skill level: Intermediate to advanced

### Core Jobs-to-be-Done

- Build a shortlist of golfers across markets quickly
- Understand why a golfer ranks highly, not just ranking position
- Reduce low-confidence bets

### Pain Points

- Fragmented data across multiple tools
- Generic pick content with weak justification
- Difficulty comparing players across contexts

### Value Drivers

- Explainable projections
- Advanced filters and comparisons
- Weekly repeatable workflow with alerts

### Conversion Triggers

- Repeated use of projection previews
- Attempts to unlock deeper model context

### Churn Risks

- Perceived lack of differentiation vs free sources
- Slow workflow or cluttered UX during peak prep windows

## Persona 2: DFS/Fantasy Optimizer

### Profile

- Age: 21-40
- Behavior: Builds multiple lineups and consumes projection data
- Skill level: Advanced

### Core Jobs-to-be-Done

- Evaluate player value relative to context
- Find leverage and differentiation opportunities
- Make final lineup decisions under time pressure

### Pain Points

- Overwhelming raw data with little synthesis
- Need to combine course fit, form, and ownership assumptions

### Value Drivers

- Dense sortable data views
- Scenario comparison workflows
- Saved presets and reusable filters

### Conversion Triggers

- Need for advanced workflow speed
- Desire for integrated DFS + betting context

### Churn Risks

- Missing advanced controls
- Inability to export or retain workflow states

## Persona 3: Research-Oriented Golf Fan

### Profile

- Age: 28-55
- Behavior: Reads tournament previews and player analysis
- Skill level: Beginner to intermediate bettor

### Core Jobs-to-be-Done

- Understand tournament context in plain language
- Learn which factors matter without becoming a quant expert
- Make occasional bets with more confidence

### Pain Points

- Intimidating analytics interfaces
- Inconsistent quality in betting media content

### Value Drivers

- Clear narrative + data hybrid content
- Progressive disclosure from simple to advanced metrics
- Trust cues and transparent assumptions

### Conversion Triggers

- Hitting limits on premium article depth
- Interest in personalized recommendations and alerts

### Churn Risks

- Overly technical experience
- Lack of onboarding guidance

## Persona 4: Time-Constrained Weekly User

### Profile

- Age: 30-50
- Behavior: Consumes product in short sessions
- Skill level: Mixed

### Core Jobs-to-be-Done

- Get decision-ready quickly
- Reuse prior week preferences with minimal setup

### Pain Points

- Long setup times before insights emerge
- Too many disconnected screens

### Value Drivers

- One-screen shortlist workflow
- Saved watchlists and preferences
- "What changed since last week" summaries

### Conversion Triggers

- Workflow acceleration features gated behind premium

### Churn Risks

- If weekly utility is not obvious by week 2-3

## Persona 5: Internal Editor/Admin

### Profile

- Role: Editorial operator or analyst
- Goal: Publish high-quality, data-grounded content efficiently

### Core Jobs-to-be-Done

- Produce timely tournament and player content
- Embed verified data and projections safely
- Maintain quality and SEO consistency

### Pain Points

- Manual research/publishing overhead
- Risk of unsupported claims during fast publishing cycles

### Value Drivers

- Structured CMS blocks
- AI assistance with grounding controls
- Template-driven publishing workflows

### Success Criteria

- Faster publishing with equal or better trust quality

## Persona Priority for MVP

1. Data-Driven Golf Bettor
2. Time-Constrained Weekly User
3. DFS/Fantasy Optimizer
4. Research-Oriented Golf Fan
5. Internal Editor/Admin

Rationale:
The first three personas are most likely to monetize quickly through premium utility. Persona 4 improves retention economics. Persona 5 is a force multiplier for content scale and quality.
