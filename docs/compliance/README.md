# Compliance and Disclosure Documentation

## Purpose

Describe the compliance, disclosure, and retention policies that govern public-facing content, operational handling, and internal review expectations for Caddy Stats.

## Scope

This folder covers:

- public legal and disclosure language
- affiliate and monetization transparency requirements
- data-retention and record-handling rules
- operational compliance expectations tied to editorial, analytics, and platform workflows

## Document map

| Document                           | Purpose                                                                                  |
| ---------------------------------- | ---------------------------------------------------------------------------------------- |
| `privacy-policy.md`                | defines how user and visitor data should be collected, processed, stored, and disclosed  |
| `terms-of-use.md`                  | defines platform usage terms, user responsibilities, and platform reservations of rights |
| `affiliate-disclosure.md`          | defines disclosure expectations for affiliate relationships and monetized content        |
| `data-retention-policy.md`         | defines retention windows, deletion expectations, and auditability requirements          |
| `operational-compliance-policy.md` | defines internal operational responsibilities around compliance-sensitive workflows      |

## How to use this folder

- update the closest policy when legal, disclosure, retention, or workflow obligations change
- cross-check these docs with `docs/security/`, `docs/data/`, and `docs/governance/` when the change affects controls or system behavior
- log material updates in `docs/changelog.md`
