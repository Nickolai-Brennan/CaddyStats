# SEO Strategy

## Objective

Grow qualified organic traffic by publishing indexable, useful golf content and entity pages that are backed by verified data and structured page systems.

## Strategic bets

- player, tournament, course, and rankings pages as evergreen entity surfaces
- editorial tournament previews and explainers as recurring search-entry content
- programmatic content only where it is supported by real entities, metadata, and differentiated value
- internal linking between entities, content, and premium surfaces to create discoverable user journeys

## Core rules

- every public page needs a unique title, description, and canonical URL
- schema.org and metadata should be treated as page contract requirements
- fast page delivery matters as much as keyword coverage
- no filler pages built from invented stats or empty templates

## Funnel role

SEO should not stop at pageviews. Public traffic should feed newsletter, account creation, and premium conversion journeys with clear value progression.

## Success measures

- organic sessions by page type
- search visibility for target golf entities and tournaments
- click-through from public pages into signup or premium exploration
- index coverage and technical health for canonical pages
